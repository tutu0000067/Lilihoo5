# Random Quote Generator


## Описание

Random Quote Generator - это десктопное приложение на Python с графическим интерфейсом для генерации случайных цитат. Приложение позволяет:

- Генерировать случайные цитаты из предопределённого списка
- Сохранять историю всех сгенерированных цитат
- Фильтровать историю по автору и теме
- Добавлять собственные цитаты с проверкой ввода
- Автоматически сохранять и загружать данные в формате JSON

## Требования

- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, random, json, os, datetime)
